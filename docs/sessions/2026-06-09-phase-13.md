## What shipped

### Pin window (always on top)
- üìç pin button in the main window header; active state shows üìå in accent color
- Calls `win.setAlwaysOnTop` via IPC; setting persists in `settings.json` and restores on launch
- "Keep window on top" On/Off toggle also in Settings > Behavior
- Help topic: `pin-window`

### Rich text editor (Tiptap)
- Replaced `<textarea>` in add/edit entry forms with a Tiptap editor
- Toolbar: Bold, Italic, Underline | Bullet list, Ordered list | Inline code, Code block | Hyperlink (prompt on click)
- Active toolbar state uses `--color-accent`
- Added `copyMode?: "plain" | "html"` to Entry type (default `"plain"`)
- "Copy as: Plain text | HTML" toggle below the editor, persists with entry
- Copy logic: plain strips HTML tags; html writes `text/html` + `text/plain` fallback via `ClipboardItem`
- MacroOverlay renders HTML bodies with `dangerouslySetInnerHTML` when `copyMode === "html"`
- Existing plain-text entries render safely
- Help topic: `rich-text`
- Packages: `@tiptap/react`, `@tiptap/pm`, `@tiptap/starter-kit`, `@tiptap/extension-link`, `@tiptap/extension-code-block`, `@tiptap/extension-underline`

### Duplicate detection on CSV import
- Preview now returns all valid rows (no 5-row cap); each row has `rowIndex`
- Duplicate rows (title + vertical, case-insensitive match against existing entries) get a Duplicate badge
- Per-row resolution dropdown: Skip (default), Overwrite, Import as new
- `commitCsvImport` accepts `resolutions: Record<number, string>` and applies them
- Default for unresolved duplicates is Skip (was previously auto-overwrite)
- `--color-warning: #f59e0b` design token added to `:root`

### Duplicate detection on manual add
- On blur of title input in add entry form, check case-insensitive title match within selected vertical
- Non-blocking amber warning below field: "An entry with this title already exists in [vertical name]."
- Warning clears when title changes; skipped in edit mode

### Stream Deck profile export
- Settings > Data > Export Stream Deck Profile
- Generates a `.streamDeckProfile` ZIP: outer manifest + page manifest with entries as buttons
- 5-column grid, capped at 32 buttons; shows save dialog
- Toast in Settings if `capped: true`
- Package: `jszip`
- Help topic: `stream-deck-export`
