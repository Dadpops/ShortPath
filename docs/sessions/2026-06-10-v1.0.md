## Decisions made

- **Entry preview positioning:** Show below the item (not to the right). The popup window is too narrow for a right-side card — the preview would land off-screen or overlap the item title. Below-the-item positioning works at any width and degrades gracefully near the bottom of the window (flips above the item).
- **Duplicate detection scope:** Detects conflicts between incoming synced entries and existing local entries only. Cross-source duplicates (two sync sources with matching entries) are not reported — that case is less actionable for the user.
- **Check-for-updates removal:** Removed entirely rather than keeping a placeholder. The `electron-updater` package is still in devDependencies (used by electron-builder) so it does not need to be removed from package.json.
